# 🌑 waiOS (Who Am I Operating System)

O **waiOS** é um ambiente de computação adaptativo baseado em Arch Linux, projetado para usuários que exigem soberania digital, alta performance e intercompatibilidade nativa.

## 🏹 Filosofia de Engenharia
O sistema é fundamentado na **Restrição Celestial**: a eliminação de telemetria e processos redundantes para priorizar o hardware bruto. Através do motor **W+A+I**, o sistema transita entre estados de consciência técnica (Perfis), otimizando o Kernel para a tarefa em execução.

## 🛠️ Especificações de Núcleo
* **Base:** Arch Linux x86-64 (Rolling Release)
* **Kernel:** Linux-Zen (Otimizado para interatividade e baixa latência)
* **Arquitetura:** Btrfs com subvolumes estruturados para resiliência de dados
* **Segurança:** Isolamento de processos e redes via Sandboxing nativo
